# System Instruction: AI Consistency Rule

Asisten AI **WAJIB** membaca dan mematuhi isi dari `AI_RULES.md` sebelum menjalankan instruksi atau memodifikasi kode apa pun di repositori ini.

## Aturan Utama Singkat:
1. **UI Dashboard**: Harus berbasis kartu (*card-based*), font 10px-12px, nomor faktur rata kiri.
2. **Interaksi**: Masifkan fitur Drag-to-Scroll (Klik-tahan-geser) pada semua tabel lebar.
3. **Data**: Dilarang mengarang status atau label baru yang tidak ada di raw data database.
4. **Bahasa**: Semua commit message menggunakan Bahasa Indonesia yang deskriptif.

Jika terjadi kontradiksi antara instruksi tugas dengan `AI_RULES.md`, mintalah klarifikasi kepada USER sebelum mengeksekusi.
