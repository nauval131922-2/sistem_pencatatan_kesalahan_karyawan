Saya baru pindah PC / membuka sesi baru. Tolong lakukan langkah berikut secara berurutan:

1. Jalankan `git pull` untuk mengambil perubahan terbaru dari remote.
   Jika ada konflik, bantu saya menyelesaikannya sebelum lanjut.

2. Baca dan ringkas isi `docs/AI_SESSION_SUMMARY.md`.

3. Baca dan tampilkan daftar task dari `docs/task.md`.

4. Tampilkan ringkasan ke saya:
   ✅ Sudah selesai  → fitur/perbaikan yang sudah di-push
   🔄 Sedang berjalan → yang belum selesai di sesi sebelumnya
   📌 Selanjutnya   → task yang perlu dikerjakan berikutnya

5. Jalankan `git status` dan tampilkan hasilnya — apakah ada file
   yang belum ter-commit dari sesi sebelumnya.

Setelah semua langkah di atas selesai, kita bisa langsung lanjut bekerja.
